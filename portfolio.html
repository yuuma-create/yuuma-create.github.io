食材・レシピをスマートに管理できるFlutter製アプリ。
Firebase連携・並び替え・タグ・画像・シェアなど、実用的な機能を多数搭載！

Designed & Developed by yuma

制作背景・目的

就職活動の準備で時間が限られる中、日常の「レシピ選びに時間がかかる」という課題を解決したいという思いから、効率よく食事を選べるように、AIを活用したレシピ管理アプリをFlutterとFirebase、GitHub Copilotを用いて個人開発しました。 技術的には、OGP解析・AIタグ生成・音声読み上げなどを実装し、忙しい日々でも素早くレシピを見つけられるよう工夫しました。

  • 制作期間：約1か月
  • 使用技術・ツール：Flutter / Dart / Firebase / Gemini API / GitHub Copilot / ChatGPT など

主な仕組み・機能

  • レシピ管理
    • レシピの追加・編集・削除（単品/一括URL追加）
    • レシピの詳細表示・WebView表示・シェア
    • レシピのタグ自動生成（Gemini API/AI連携）
    • レシピのクラウドリアルタイム同期（Firestore）
  • 食材管理
    • 食材のカテゴリ分け・画像管理（ローカル/URL/プリセット画像）
    • 食材の在庫・賞味期限・数量管理
    • 食材の検索（漢字/かな/カナ/ローマ字対応）
  • 献立提案
    • 1週間分の献立をAIが自動提案（材料・レシピ・オプション指定可）
    • 提案結果の保存・再表示
  • 検索・その他
    • 複数サイト検索（WebView）
    • 音声読み上げ（TTS）
    • 材料自動計算（倍率指定で材料量を自動変換）
    • メモ機能
    • 設定画面（動作確認済みサイト一覧、ライセンス表示、プライバシーポリシー等）
    • ユーザー認証（Google/Apple/Firebase）

代表的なコード例

レシピの賞味期限表示ロジック（utils.dart）

class ExpiryInfo {
  final int displayDays;
  final Color expiryColor;
  final String expiryText;
  ExpiryInfo(this.displayDays, this.expiryColor, this.expiryText);
}

ExpiryInfo getExpiryInfo(int? daysUntilExpiry) {
  int displayDays = daysUntilExpiry ?? 999;
  Color expiryColor = Colors.grey;
  String expiryText = "";
  if (displayDays < 0) {
    expiryColor = Colors.red.shade700;
    expiryText = "(期限切れ)";
  } else if (displayDays == 0) {
    expiryColor = Colors.red;
    expiryText = "(あと0日)";
  } else if (displayDays <= 3) {
    expiryColor = Colors.red;
    expiryText = "(あと $displayDays 日)";
  } else if (daysUntilExpiry != null) {
    expiryColor = themeColorNotifier.value;
    expiryText = "(あと $displayDays 日)";
  }
  return ExpiryInfo(displayDays, expiryColor, expiryText);
}

自己PR・エピソード

私の強みは「自分で課題を見つけて、手を動かしながら解決する行動力」と「粘り強く学び続ける姿勢」です。
小学生の頃に中古ノートPCの性能改善に挑戦したことをきっかけに、ハードウェアへの関心が高まりました。高校時代には古物商許可を取得し、ジャンクPCを修理・販売する中で、調査・実践・改善を繰り返し、年間30万円前後の売上を安定して出すことを目標に運営していました。

現在は、FlutterとGitHub Copilotを活用し、自炊生活で感じた「レシピ選びの手間」を解消するアプリを開発中です。AIによるレシピ提案や検索機能を実装し、iOS・Androidの両方で実用的に動作するレベルまで完成させました。これらの経験を通じて、必要とされるものを考え、実装する力や、粘り強く学ぶ姿勢を培いました。今後は、これらの力をチーム開発でも活かし、設計やレビューを通じて成長していきたいです。

個人開発する中で、生成AIや機械学習の可能性に魅力を感じ、日常の課題を技術で解決できる面白さを実感しました。今後は、AIをはじめとする新しい技術への理解をさらに深め、より多くの人の役に立つサービスづくりに取り組んでいきたいと考えています。また、これまでは一人での開発が中心でしたが、今後はチームでの開発にも積極的に挑戦したいです。設計や分担、レビューなどを通して、実務に必要な技術力はもちろん、コミュニケーション力や課題整理力といったチームで働く力も高め、より実践的なスキルを身につけていきたいと考えています。
