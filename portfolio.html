Memoia - AI駆動型メモアプリ

AIが目的を推測し、最適なUI・構造を自動生成する次世代メモアプリ

アプリを試す

プロジェクト概要

Memoiaは、メモの内容をAIが分析し、テーブル形式やリスト形式など最適なレイアウトを提案する次世代メモアプリです。 画像OCR、AIチャット、Stripe決済デモなど、30以上の機能を実装。Vercelで本番公開中です。

💡 開発の動機

既存のメモアプリは「書く」ことに特化していますが、「書いた内容をどう整理するか」はユーザー任せでした。 AIの力を使えば、入力内容から意図を理解し、最適な構造を自動提案できるのではないかと考えました。

Next.js 14とマルチAI APIを統合し、実用的なプロダクトとして本番公開まで実現しました。

主な機能

🖼️ 画像OCR

写真から構造化されたメモを自動生成。Google Cloud Vision APIで高精度な文字認識を実現。

🤖 AIチャット

複数メモを参照した高度な質問対応。Gemini/ChatGPT APIによるマルチAI統合。

📊 テーブル自動生成

テキストから構造化されたテーブルに自動変換。列・行の追加削除、ドラッグリサイズに対応。

💳 Stripe決済デモ

サブスクリプション課金システム（テストモード）。Webhook連携、顧客管理の実装デモ。

📝 リッチエディタ

直感的な編集体験。ブロック単位での管理、マークダウン対応、テンプレート機能。

☁️ リアルタイム同期

Firebase Firestoreで複数デバイス間で即座に反映。クラウドベースでどこからでもアクセス可能。

💡 スマートレイアウト

AIが内容に応じて最適なUIを提案。リスト/テーブル/段落を自動判定。

🌐 本番公開

VercelでCI/CD完備。カスタムドメイン、SSL対応、グローバルCDNで高速配信。

使用技術・ライブラリ

⚛️ Next.js 14

App Router、Server Components、Server Actionsなど最新機能を活用したモダンなアーキテクチャ。

📘 TypeScript

型安全な開発による堅牢性の確保。カスタム型定義で保守性向上。

🔥 Firebase

Firestore Database、Authentication、Hostingを統合。リアルタイムデータ同期を実装。

🤖 AI API統合

Gemini API・ChatGPT API両対応。高精度なテキスト分析とコンテンツ生成。プロンプトエンジニアリングを実践。

💳 Stripe (テストモード)

サブスクリプション課金システムのデモ実装。Webhook連携、顧客管理機能を検証。

▲ Vercel

CI/CD、プレビューデプロイ、グローバルCDN。本番環境の構築・運用を実現。

🎨 Tailwind CSS

ユーティリティファーストなスタイリング。レスポンシブデザインを効率的に実装。

🎯 React Hooks

カスタムフックによる状態管理とロジックの分離。useContext、useMemo、useCallbackを活用。

制作背景・目的

既存のブラウザアプリに満足できず、「自分が本当に使いたい機能を持つブラウザを作りたい」という思いから開発を開始しました。 特に、マルチタブ管理の効率性、豊富なカスタマイズ機能、プライバシー保護機能、そして直感的な操作性を重視し、 ユーザーが快適にWebブラウジングできる環境を目指しました。Flutter の WebView を活用し、ネイティブアプリとしての性能も確保しています。

制作期間：3週間

開発形態：個人開発

使用技術：Flutter / Dart / WebView / SharedPreferences / Material Design

プロジェクトについて

本プロジェクトは個人の学習・研究目的で開発したプロトタイプです。 データ分析、可視化技術、GUI設計の習得を目的としており、実運用は行っておりません。 サーバーに負荷をかけない設計を心がけ、技術的な学びに焦点を当てた学習・研究プロジェクトです。

プロジェクト概要

複数のデータソースから価格情報を収集し、統合的に分析・可視化するPythonアプリケーションです。 古物商としての実務経験を活かし、AIの力を借りながら開発した個人学習プロジェクトです。

💡 開発の動機

古物商として実際に商品の仕入れ・販売を行う中で、「目利き」や「市場の読み」といった 経験則に頼っていた部分を、データとロジックに基づいてIT化できないかと考えました。

AIの力を借りながら、実務で直面する課題をプログラミングで解決する経験を通じて、 データ収集・分析・可視化の基礎技術を学ぶことができました。

技術的特徴

🔍 データ収集の自動化

Seleniumを使用したWebデータの自動取得システム。動的ページの処理、待機戦略、エラーハンドリングを実装。

📊 データ分析基盤

Pandasによるデータクレンジング、集計、統計分析。複数データソースの統合とクロス集計の実装。

📈 可視化システム

Matplotlibで4種類のグラフを実装。価格分布、プラットフォーム比較、利益マップ、ランキング表示。

🖥️ GUIアプリケーション

tkinterによる直感的なインターフェース設計。タブ構成、リアルタイム更新、フィルタリング機能を実装。

💾 データ永続化

CSV形式でのデータ保存・読み込み。キーワードリストのファイル管理システムを構築。

🔧 設計パターンの習得

スクレイパーの基底クラス設計、継承による拡張性、モジュール分離などOOP設計を実践。

実装した機能

📋 一括データ収集

複数キーワードの一括処理、プラットフォーム横断検索、取得件数の柔軟な設定が可能。

💰 価格分析機能

平均価格算出、価格差の抽出、利益率計算、最安値・最高値の自動検出を実装。

🎯 高度なフィルタリング

プラットフォーム選択、価格範囲指定、データ件数による絞り込みが可能。

📊 4種類のグラフ

プラットフォーム比較、利益マップ、価格分布ヒストグラム、Top10ランキング。

📁 ファイル管理システム

キーワードリストのフォルダ管理、ファイル作成・編集・保存の統合インターフェース。

💾 データ保存・読み込み

検索結果のCSV出力、過去データの再読み込み、分析結果の永続化。
