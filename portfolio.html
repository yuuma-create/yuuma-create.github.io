就活効率化アプリ

就職活動を効率化するオールインワンアプリケーション

制作背景・目的

就職活動を始めた際に、企業情報の管理、選考進捗の把握、ES（エントリーシート）の作成・管理など、多くのタスクを別々のツールで管理する煩雑さを感じました。これらの課題を解決し、就活生が効率的に活動できるよう、全ての機能を統合したFlutterアプリを個人開発しました。

制作期間：約2ヶ月

開発形態：個人開発

使用技術：Flutter / Dart / Provider / Material Design / Local Storage

主な機能・特徴

🏢 企業管理

企業情報の登録・編集、業界分類、企業規模、勤務地、給与情報などの詳細管理

📊 選考進捗管理

ES提出、筆記試験、面接など各選考段階の進捗状況をタイムライン形式で管理

📝 ES管理

エントリーシートのテンプレート作成、企業別カスタマイズ、文字数カウント、データエクスポート

📅 スケジュール管理

説明会、面接、インターンなどの予定をカレンダー形式で管理、リマインダー機能

📈 統計・分析

応募数、通過率、業界別分析など就活状況をグラフで可視化、改善点の発見

🔍 企業検索

業界、規模、勤務地など多様な条件での企業検索、お気に入り登録機能

Flutter レシピ管理アプリ

AIを活用した次世代レシピ管理アプリケーション

制作背景・目的

就職活動の準備で時間が限られる中、日常の「レシピ選びに時間がかかる」という課題を解決したいという思いから、効率よく食事を選べるように、AIを活用したレシピ管理アプリをFlutterとFirebase、GitHub Copilotを用いて個人開発しました。

制作期間：約1ヶ月

開発形態：個人開発

使用技術：Flutter / Dart / Firebase / Gemini API / GitHub Copilot / ChatGPT

主な機能・特徴

📝 レシピ管理

レシピの追加・編集・削除、WebView表示、シェア機能、AIタグ自動生成、クラウドリアルタイム同期

🥬 食材管理

カテゴリ分け、画像管理、在庫・賞味期限・数量管理、多言語対応検索機能

🤖 AI献立提案

1週間分の献立をAIが自動提案、材料・レシピ・オプション指定可能、提案結果の保存・再表示

🔍 高度な検索

複数サイト検索（WebView）、漢字/かな/カナ/ローマ字対応、音声読み上げ（TTS）

🧮 材料計算

倍率指定で材料量を自動変換、メモ機能、設定画面（ライセンス表示、プライバシーポリシー等）

🔐 認証機能

Google/Apple/Firebase認証、ユーザーデータの安全な管理

技術スタック

Flutter Dart Firebase Firestore Firebase Auth Gemini API TTS WebView Provider GitHub Copilot
